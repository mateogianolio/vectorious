Vector API

Vector API


A vector is a one-dimensional representation of data, analogous to an array. Under the hood it is by default represented by a Float64Array.

A Vector can be created with a regular array:

var vector = new Vector([1, 2, 3]);


or with a TypedArray:

var data = new Uint8Array([1, 2, 3]);
var vector = new Vector(data);
// vector is now of type Uint8Array


or with a Matrix:

var matrix = new Matrix([[1], [2], [3]]);
var vector = new Vector(matrix);
// vector is now a two-dimensional representation of the matrix


The following vector operations and methods are implemented in vector.js:


zeros ones range random

add subtract scale normlize dot magnitude angle project min max equals get set combine push

add


// (Vector, Vector) => (Vector)
Vector.add = function(a, b)

// (this, Vector) => (this)
Vector.prototype.add = function(vector)

Add two vectors together.


var a = new Vector([1, 2, 3]),
b = new Vector([4, 5, 6]);

Vector.add(a, b);
// [5, 7, 9]

a.add(b);
// a := [5, 7, 9]

// (Vector, Vector) => (Vector)

Vector.prototype.subtract = function(vector)

b = new Vector([4, 5, 6]);

Vector.subtract(a, b);

a.subtract(b);
// a := [-3, -3, -3]

zeros


// (Number) => (Vector)
Vector.zeros = function(count, [type])

Create a vector of count zeros. type is optional and specifies the type of TypedArray used in computations.


Vector.zeros(5);
// [0, 0, 0, 0, 0]

ones


// (Number) => (Vector)
Vector.ones = function(count, [type])

Create a vector of count ones. type is optional and specifies the type of TypedArray used in computations.


Vector.ones(5);
// [1, 1, 1, 1, 1]

range


// (Number, [Number], Number) => (Vector)
Vector.range = function(start, [step], end, [type])

Create a vector containing the range from start to end in steps of step (optional). type is optional and specifies the type of TypedArray used in computations.


Vector.range(0, 5);
// [0, 1, 2, 3, 4]

Vector.range(0, 0.5, 2);
// [0, 0.5, 1, 1.5]

Vector.range(5, 0);
// [5, 4, 3, 2, 1]

Vector.range(2, 0.5, 0);
// [2, 1.5, 1, 0.5]

random


// (Number) => (Vector)
Vector.random = function(count, [deviation], [mean], [type])

Create a vector of count elements containing random values according to a normal distribution of standard deviation and mean. type is optional and specifies the type of TypedArray used in computations.


Vector.random(3);
// e.g. [0.9449769428465515, 0.07831531320698559, 0.15513482666574419]

Vector.random(3, 4, -4);
// e.g. [-0.9220099672675133, -0.20406165812164545, -3.104398760944605]

equals


// (Vector, Vector) => (Boolean)
Vector.equals = function(a, b)

// (this, Vector) => (Boolean)
Vector.prototype.equals = function(vector)

Compare two vectors.


var a = new Vector([1, 2, 3]),
b = new Vector([4, 5, 6]);

Vector.equals(a, b);
a.equals(b);
// false

b = new Vector([1, 2, 3]);

Vector.equals(a, b);
a.equals(b);
// true

// (this, Function) => (Vector)
Vector.prototype.map = function(callback)

Maps a function callback(value, index) to all elements of the vector.

// (this, Function) => (this)
Vector.prototype.each = function(callback)
