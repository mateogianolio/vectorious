Matrix API

Matrix API


A matrix is a two-dimensional representation of a vector stored in a one-dimensional array. For performance reasons the data is stored by default in a Float64Array.

You can create a matrix by passing a two-dimensional array to the Matrix constructor:

var matrix = new Matrix([[1, 2], [3, 4]]);


or with a TypedArray:

var data = new Uint8Array([1, 2, 3, 4]),
shape = [2, 2];

var matrix = new Matrix(data, { shape: shape });
// matrix type is now Uint8Array


You can also create a matrix with a Vector, as long as you supply the desired matrix shape:

var vector = new Vector([1, 2, 3, 4]),
shape = [2, 2];

var matrix = new Matrix(vector, { shape: shape });


The following matrix operations and methods are implemented in matrix.js:

gauss


// (this) => (Matrix)
Matrix.prototype.gauss = function()

Convert a matrix to reduced row echelon (RREF) form using Gauss-Jordan eliminiation..


var a = new Matrix([[1, 2], [3, 4]]);

a.gauss();
// [[1, 0], [0, 1]]

lu


// (this) => ([Matrix, Matrix, Int32Array])
Matrix.prototype.lu = function()

Get the full LU decomposition of a matrix in the form of the array [L, U, P], where P an Int32Array of the pivoted row indices.


var a = new Matrix([[1, 2], [3, 4]]);

a.lu();
// L: [[1, 0], [0.333., 1]]
// U: [[3, 4], [0, 0.666.]]
// P: [1, 1]

plu


// (this) => ([Matrix, Int32Array])
Matrix.plu = function(matrix)

// (this) => ([this, Int32Array])
Matrix.prototype.plu = function()

Get the LU factorization of a matrix in the form of the array [LU, P], where P an Int32Array of the pivoted row indices and LU is the current matrix.


var a = new Matrix([[1, 2], [3, 4]]);

a.plu();
// LU: [[3, 4], [0.333., 0.666.]]
// P: [1, 1]

lusolve


// (this, Matrix, Int32Array) => (Matrix)
Matrix.prototype.lusolve = function(rhs, ipiv)

Solves an LU factorized matrix this with the supplied right hand side(s) rhs.


var a = new Matrix([[1, 3, 5], [2, 4, 7], [1, 1, 0]]),
rhs = new Matrix([[1], [3], [5]]);

var plu = a.plu(),
lu = plu[0],
ipiv = plu[1];

lu.lusolve(rhs, ipiv);
// rhs: [[3.25], [1.75], [-1.5]]

solve


// (this, Matrix) => (Matrix)
Matrix.prototype.solve = function(rhs))

Solves AX = B using LU factorization, where A is the current matrix this and B is a Vector/Matrix containing the right hand side(s) rhs of the equation.


var a = new Matrix([[1, 3, 5], [2, 4, 7], [1, 1, 0]]),
rhs = new Matrix([[1], [3], [5]]);

a.solve(rhs);
// [[3.25], [1.75], [-1.5]]

each


// (this, Function) => (this)
Matrix.prototype.each = function(callback)

Calls callback(value, i, j) for each element in the matrix.

}

var a = new Matrix([[1, 2], [3, 4]]);

a.each(log);
// 1 0 0
// 2 0 1
// 3 1 0
// 4 1 1
